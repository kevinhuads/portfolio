Deep learning · MLOps

End to end deep learning computer vision workflow

Multi-class image classification on the Food-101 dataset, from exploratory analysis to deployment, with a production-oriented structure.

  • Systematic benchmark of modern CNN and transformer architectures, under frozen and fine-tuning regimes.
  • Training engine with mixed precision, OneCycleLR and detailed metrics, including calibration and per class diagnostics.
  • Experiment tracking with MLflow and structured artifacts: checkpoints, CSV logs and diagnostic plots.
  • Streamlit application for inference, packaged with Docker and integrated in a GitHub Actions CI/CD workflow.
  • Test suite covering data pipeline, metrics, CLIs and a synthetic end to end training run.
